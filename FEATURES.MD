Интерфейс модулей пакета и документация написаны на русском и английском
языках. Перевод на другие языке не предусмотрен, но возможен.

## ФУНКЦИИ АДМИН ПАКА

 - Работает с русскими версиями Microsoft Windows и Microsoft Windows Server

 - Работает на компьютерах начиная с версии Microsoft Windows XP SP3

 - В своей работе требует права локального администратора либо администратора домена (в зависимости от версии Windows)

 - Устанавливается в каталог %SystemRoot% Microsoft Windows

 - Может использоваться для первоначальной настройки системного и сетевого окружения при первоначальной установке компьютеров Microsoft Windows на предприятии

 - Может использоваться при установке компьютеров с Microsoft Windows на удалённом хостинге

 - Может использоваться при восстановлении оптимального состояния компьютеров и сети после сбоев и вирусной атаке

 - Может использоваться в повседневной работе Windows администратора

 - Несколько вариантов для установки пакета

## ДОПОЛНИТЕЛЬНО

Для работы всех функций и установки всех приложений к компьютеру/серверу и всем другим компьютерам в сети должен быть открыт доступ из интернета. Для работы основных функций такого доступа не нужно.

Является частью системного окружения пакета NIT Projects Environment, но им по умолчанию не устанавливается.
