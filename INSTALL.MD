# INSTALL ADMIN PACK
(ИНСТАЛЛЯЦИЯ АДМИН ПАКА)

Поскольку данный Админ Пак изготовлен в виде командных файлов, конфигурировать и компилировать файлы не нужно. После установки в целевой каталог Админ пак готов к работе.

Админ пак устанавливается в каталог Windows %SystemRoot% (по умолчанию это C:\Windows)

Существует несколько способов установки Админ Пака на компьютер с операционной системой Microsoft Windows, различающихся простотой подготовки файлов и их установки. Перечислим все из них.

## ПОДГОТОВКА — УСТАНОВКА ИСХОДНИКОВ С ГИТХАБА

- Создайте и перейдите в каталог %TEMP%\Scripts командами

    `md %TEMP%\Script`
    `cd /d %TEMP%\Script`

- Клонируйте в него указанный репозиторий командой

    `git clone http://github.com/yudenisov/Scripts ???`

или просто разархивируйте туда архив, скаченный с GitHub

- Установите через менеджер пакетов Chocolatey пакет InnoSetup и WinRAR:

    `choco install -y innosetup winrar`

## ПЕРВЫЙ СПОСОБ УСТАНОВКИ

Просто скопируйте содержимое каталога %TEMP%\Script\Scripts\Files вместе с подкаталогами в каталог %SystemRoot%. Однако учтите, что в этом случае удалять админ-пак Вы будете вручную, отыскивая и удаляя каждый файл и зависимость.

## ВТОРОЙ СПОСОБ УСТАНОВКИ (ОРИГИНАЛЬНЫЙ)

- Запустите файл %TEMP%\Script\Scripts\install.bat

- Для создания самораспаковывающегося архива запустите этот же файл install с ключом *-sfx*

- Для удаления установленного Админ пака запустите этот же файл install с ключом *-u*

## ТРЕТИЙ СПОСОБ УСТАНОВКИ (ПРАВИЛЬНЫЙ)

- Откройте файл %TEMP%\Script\Scripts\Files\AdminScripts.iss и внесите в него необходимые минимальные изменения

- Скомпилируйте этот файл

- Получится файл AdminScriptsSet01.exe

Этот файл является полноценным инсталлятором. С его помощью можно установить приложение, и инсталлятор автоматически пропишет правила его корректного удаления.
